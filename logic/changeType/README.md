###类型转换节点
本节点用于将msg.payload的数据类型改变为整型或字符串。

####参数配置
<li>Name：节点名称</li>
<li>Operation：转换类型</li>

####输入量
<li>msg.payload：输入转换前的数据</li>

####输出量
<li>msg.payload：输出转换后的数据</li>

####使用方法
<li>配置好节点的每个参数。</li>
<li>输入量<code>msg.payload</code>的值为整型数或由数字组成的字符串。</li>
###Change-type Node
This node is used to change the data type of the msg.payload into integer or string.

####Parameter Configuration
Name: node name
Operation: The change type

####Input
msg.payload：The data before changing type
####Output
msg.payload: The data after changing type

####Howto
Configure every parameter correctly.
The input number <code>msg.payload</code> should be the string consist of number or integer number.

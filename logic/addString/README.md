###增加字符串节点

本节点可以允许用户在msg.payload前后添加字符串。

####配置参数
<li>Name：节点名称</li>
<li>Prepend：前缀字符串</li>
<li>Append：后缀字符串</li>

####输入
<li>msg.payload：输入字符串</li>

####输出
<li>msg.payload：拼接后的字符串</li>

####使用方法
<li>配置好节点的每个参数。</li>
<li><code>msg.payload</code>可以为字母，数字，中文或者特殊字符组成的字符串。</li>


###Add string node

This node allow user add string before/after the msg.payload.

####Config
<li>Name：Name</li>
<li>Prepend：Prepend string</li>
<li>Append：Append string</li>

####Input
<li>msg.payload: Input string</li>

####Output
<li>msg.payload：appended string</li>

####Howto
<li>configurate every parameter in the node</li>
<li><code>msg.payload</code> cna be a string which is composed of the number, character, chinaese or Special character.</li>

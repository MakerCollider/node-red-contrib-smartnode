###舵机节点
本节点表示Arduino套件的舵机模块。

####参数配置
Name：节点名称
Board：开发板选择
Digital Pin: 输入引脚
startAt：开始角度

####输入量
msg.payload：输入1或者0
msg.payload：输入旋转角度

####输出量
无

####使用方法
配置好节点的每个参数。
###Servo Node
This node represents Servo module in Arduino kit.

####Parameter Configuration
Name: node name
Board: arduino board select
Digital Pin: input pin
startAt: start angle

####Input
msg.payload：The input rotation angle

####Output
NULL

####Howto
Configure every parameter correctly.
